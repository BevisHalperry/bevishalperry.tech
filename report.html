State 3 things you had to learn/find out by yourself when creating your site, and describe how you did that (i.e. what searches you did, any new tools/techniques you learned, any changes/adaptations you made to a particular resource to make it work with your site).

  • Advanced Styling: I learned LESS, the css preprocessor. I usually use SASS, but wanted to try something new and liked I could use it with out any build tools as it can compile client-side using less.js, very useful for low-key devlopment. I just used the LESS docs to learn. I tried to learn the BEM methodology, and incorperate a few of the rules into the LESS, it helped to structure a lot of the styling and make it very DRY, I can see this method being very useful for large projects.
  • Accessability: I learned how tabindex works for screen readers and how to make it work well within a page, its not always obvious and a lot of browsers have strange default behavior, like tabindex always starting with the input forms. Found that tabindex has to be daclared in the parent object to override this bug
  • Forms: I learned how Formspree works and how to implement it. Its incredibly well built and very easy to implement. I also read about how to host your own Formspree server.
